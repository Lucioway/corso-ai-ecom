01 · Cos'è
図譜
github.com/safishamsi/graphify · open source · gira in locale

Cos'è Graphify.

Cos'èUn tool open source che legge i tuoi file markdown e ne disegna la mappa.
A cosa serveClaude interroga la mappa invece di rileggere tutti i file: risposte in secondi.
Come si usaInstalli, punti una cartella, apri il grafo nel browser. Tre comandi.
Elemento · 01
Nodo
Un concetto trovato nei tuoi file. Ogni puntino della mappa: "pricing", "offerta", "garanzia".
Elemento · 02
Arco
Un collegamento tra due concetti. Ogni linea: "pricing si lega a valore percepito".
Elemento · 03
Mappa
Il grafo completo: navigabile nel browser, interrogabile da Claude via MCP.

I tuoi file sono un libro senza indice. Graphify costruisce l'indice intelligente.

Blocco 01Nodi · archi · mappa
02 · Perché
理由
L'idea di Karpathy: non documenti, una rete di concetti.

Elenco = cerchi tu. Mappa = chiedi e basta.

Beneficio · 01
Risparmio token
Claude legge 3 nodi invece di 300 file. Risposte più veloci, contesto pulito, costo a una frazione.
Beneficio · 02
Collegamenti nascosti
Il grafo vede relazioni tra documenti lontani — anche quelle che tu non ricordavi di aver letto.
Beneficio · 03
Risposte citate
Ogni nodo punta al file sorgente. Claude ti dice da quale video viene l'informazione. Verificabile.

Più i concetti si incrociano, più la mappa diventa ricca. Hormozi è il caso perfetto.

04 · Setup
設置
Nessuna configurazione. Nessun server. Nessun abbonamento.

3 comandi e il grafo nasce.

# 1 — uv, il gestore pacchetti (se manca)
curl -LsSf https://astral.sh/uv/install.sh | sh
 
# 2 — installa Graphify (pacchetto: doppia y!)
uv tool install graphifyy
 
# 3 — verifica (comando: una y sola)
graphify --help

Poi in Claude Code: "costruisci il grafo della cartella Brains/Hormozi" — 1–3 minuti su un canale intero.

Tre test, dal facile al difficile.

Query · 01

"Quali sono i concetti più collegati?" → offer, value, pricing. I pilastri del suo pensiero, trovati matematicamente.
Query · 02
Il percorso
"Che collegamento c'è tra pricing e garanzia?" → il path in 2-3 salti. Nessuna ricerca per parola chiave può farlo.
Query · 03
La domanda secca
"Cosa dice Hormozi sugli sconti?" → risposta in secondi, con i file citati. Apri la fonte e verifichi.

Per rispondere ha letto 3 nodi, non 300 file. Questo è il motivo per cui si costruisce il grafo.
