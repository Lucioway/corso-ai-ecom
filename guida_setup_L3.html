01 · La mappa

Cosa installiamo e perché

Tre programmi distinti, stesso Claude dentro. Cambia solo da dove ci parli. Li installi tutti e tre, poi usi quello giusto per il momento giusto.

Vive nel terminale. Legge file, esegue comandi, lancia agent. Tutto il sistema gira qui.

02 · Fondamenta · Mac

Homebrew + Node

Homebrew = l'app store del terminale Mac. Node.js = il runtime su cui gira Claude Code. Due comandi, copia e incolla nel Terminale (Applicazioni → Utility → Terminale).

# 1. Homebrew (salta se ce l'hai già)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 2. Node.js
brew install node

# 3. Verifica — devono uscire due numeri di versione
node -v   # es. v22.x
npm -v    # es. 10.x
Se i due numeri escono, le fondamenta sono pronte. Se qualcosa si rompe: copia l'errore esatto e incollalo a Claude in chat — te lo risolve lui.
03 · Il motore

Claude Code CLI + login

Il CLI ufficiale di Anthropic: il motore di tutto. Si installa con npm, poi claude apre il browser per il login.

# Installa (globale: lo lanci da qualsiasi cartella)
npm install -g @anthropic-ai/claude-code

# Verifica
claude --version

# Avvia — si apre il browser per il login Anthropic
claude

# Primo test, dentro la sessione
› ciao, chi sei?
Se risponde, sei dentro. Se il login non parte da solo: claude /login e segui il link.
04 · Daily driver

Antigravity (IDE)

L'IDE di Google: editor, browser e terminale in una finestra. Il CLI di Claude gira nel suo terminale integrato.

1
Scarica + trascinaVai su antigravity.google, scarica il .dmg, trascina l'app in /Applicazioni. Come ogni app Mac.
2
Apri una cartellaFile → Open Folder → la cartella del progetto (o il vault). Antigravity lavora sempre dentro una cartella.
3
Apri il terminale integratoPremi Ctrl + ` (backtick). Si apre il terminale dentro l'IDE — stessa shell del Mac.
4
Lancia il motoreScrivi claude nel terminale integrato. Il CLI parte lì dentro.
Antigravity = la macchina. Claude Code CLI = il motore. Stesso Claude sotto, ma da qui ci lavori tutto il giorno.

06 · way-stack

way-stack — il mio stack in 3 comandi

Il mio Claude Code stack pubblicato su GitHub: skill, agent, hook, orchestrator, vault. Da dentro Claude Code (CLI aperto):

# 1. Aggiungi il marketplace
› /plugin marketplace add Lucioway/way-stack

# 2. Installa il plugin
› /plugin install way-stack

# 3. Copia il CLAUDE.md nella root del tuo progetto
› copia il CLAUDE.md di way-stack nella root di questo progetto

Da quel momento hai GSD, BMAD, gstack, superpowers, frontend-design e tutti gli skill pre-cablati. Repo + README: github.com/Lucioway/way-stack

6 mesi del mio lavoro, 3 comandi. Non c'è motivo di partire da zero quando puoi partire da qui.
07 · La sessione

Come si lavora in Antigravity

Una sessione = una cartella aperta in Antigravity + la CLI di Claude che gira nel suo terminale. Quella cartella è il contesto: Claude legge il CLAUDE.md, capisce dove sei, lavora lì dentro.

passo · 01
Apri la cartella
File → Open Folder → il vault o il progetto su cui lavori.
passo · 02
Lancia claude
Terminale integrato (Ctrl + `) → claude. Parte la sessione, legge il CLAUDE.md.
passo · 03
Chiedi · leggi · approva
Lui propone, tu leggi il diff, approvi o correggi. Mai accettare alla cieca.

Contesto pieno o cambi task? /clear azzera e riparti pulito. Chiudi il terminale = chiudi la sessione; riaprilo e claude per riprendere. Cambi progetto = apri un'altra cartella, nuova sessione.

09 · Windows

Su Windows — WSL2

Claude Code richiede WSL2 (un Linux dentro Windows), non Windows nativo. Antigravity e Desktop hanno build Windows native; la CLI sta dentro WSL2.

# In PowerShell come amministratore
wsl --install

# Riavvii, apri Ubuntu, poi:
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install nodejs
npm install -g @anthropic-ai/claude-code
claude
In Antigravity, quando apri il terminale integrato scegli WSL come shell. Guida ufficiale aggiornata: docs.claude.com/en/docs/claude-code
