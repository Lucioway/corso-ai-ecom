Blocco 01La mappa · 3 pezzi
02 · Fondamenta
⌘
Prima del motore servono le fondamenta.

Homebrew + Node.

Cos'èHomebrew = l'app store del terminale Mac. Node.js = il runtime su cui gira Claude Code.
A cosa serveCon Homebrew installi qualunque tool in un comando. Node serve a far girare il CLI.
Come si usaDue comandi, copia e incolla. Poi verifichi con node -v. Fine.
— zsh — Terminale —
# 1. Homebrew (se non ce l'hai già)
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
# 2. Node.js
$ brew install node
# 3. Verifica — devono uscire due numeri di versione
$ node -v   # v22.x
$ npm -v    # 10.x ▌

Se i due numeri escono, le fondamenta sono pronte. Se qualcosa si rompe: incolla l'errore a Claude in chat — te lo risolve lui.

Blocco 04Antigravity · IDE~3 min
04 · La sessione
作業
Apri cartella, lanci claude, parli a parole.

Come si lavora.

Cos'èUna sessione = una cartella aperta in Antigravity + la CLI di Claude che gira nel suo terminale.
A cosa serveQuella cartella è il contesto. Claude legge il CLAUDE.md, capisce dove sei, lavora lì dentro.
Come si usaFile → Open Folder → Ctrl + ` → claude. Cambi progetto = apri un'altra cartella.
Passo · 01 — apri
Apri la cartella
File → Open Folder → il vault o il progetto. Antigravity lavora sempre dentro una cartella.
Passo · 02 — avvia
Lancia claude
Terminale integrato (Ctrl + `). Parte la sessione: Claude legge il CLAUDE.md e capisce il contesto.
Passo · 03 — il ciclo
Chiedi · leggi il diff · approvi
Lui propone le modifiche, tu leggi il diff, approvi o correggi. Mai accettare alla cieca.

Contesto pieno o cambi task? /clear azzera e riparti pulito.  ·  Chiudi il terminale = chiudi la sessione. Riapri + claude = riprendi.

08 · Windows
⊞
窓
Funziona su Windows, ma passa da Linux.

Windows · WSL2.

Cos'èWSL2 = un Linux che gira dentro Windows. Claude Code richiede WSL2 (non Windows nativo).
A cosa serveFar girare il terminale Unix che Claude Code si aspetta. Stessa esperienza del Mac.
Come si usaUn comando PowerShell, riavvio, poi gli stessi passi del Mac dentro Ubuntu.
# In PowerShell come amministratore
> wsl --install
# Riavvii, apri Ubuntu, poi:
$ curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
$ sudo apt install nodejs
$ npm install -g @anthropic-ai/claude-code
$ claude

Io uso Mac e Windows non lo padroneggio. Per la guida ufficiale aggiornata → docs.claude.com/en/docs/claude-code
