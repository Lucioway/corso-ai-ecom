04 · Theme CLI
主題
Il tema non è più una scatola nera: è una cartella di file.

Il tema diventa file.

pull
Scarica
shopify theme pull — il tema in locale: sections, templates, assets.
dev
Anteprima
shopify theme dev — preview live che si aggiorna mentre modifichi.
push
Deploy
shopify theme push — carica le modifiche sullo store.

Tutto quello che è un file, Claude lo sa leggere e modificare. E mai sul tema live: si duplica, si lavora sul dev theme.
